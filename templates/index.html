API Documentation
Endpoints
GET /api/transcript/<video_id>

Extract transcript by video ID

POST /api/transcript

Extract transcript by URL (first 20 minutes)

{
  "url": "https://youtube.com/..."
}
Response Format
  • video_id - YouTube video ID
  • language_code - Detected language (e.g., "en", "es")
  • language_type - "en" or "non-en"
  • full_text - Complete transcript text
  • captions - Array of timed segments
  • video_metadata - Title, description, etc.
Features
  • Automatic language detection
  • Clean text processing
  • Removes filler content ([Music], etc.)
  • Metadata extraction via HTML scraping
  • Proxy support for metadata extraction
  • Graceful error handling
Configuration

Proxy Setup (Optional):

Set environment variable for proxy:

METADATA_PROXY_URL=http://proxy:port

Used only for metadata extraction if YouTube is blocked
